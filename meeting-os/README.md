# Meeting OS

Bookend any meeting: build a tight agenda before it and a follow-up email
plus clean action list after it. Drafts only — it never sends.

Built by MBL Partners (https://mblpartners.com) — part of the Thrive with AI
skill library.

---

## What It Does

Detects the mode automatically and operates in two phases:

**Agenda Mode (before a meeting)** — takes a calendar invite, topic, or rough
notes and produces a focused agenda that forces the meeting down to the few
decisions that justify it. After presenting the agenda, it challenges it once:
if there is no real decision, it says so. The best outcome is sometimes "this
could be an email," and it says that when it is true.

**Follow-Up Mode (after a meeting)** — takes your notes or transcript and
produces two artifacts: a ready-to-send recap email with decisions and action
items, and a clean action list you can paste directly into your task tool.

Both outputs are drafts. The skill never sends anything.

## Install

1. Download `meeting-os.skill`
2. Open Cowork
3. Drag the `.skill` file into your Cowork window
4. The skill installs automatically and is ready to run

---

## How to Run It

Once installed, trigger it by saying any of the following:
- "Build the agenda"
- "Meeting agenda"
- "Write the follow-up"
- "Meeting recap"
- "Run meeting OS"

If it's unclear which mode you need, the skill will ask.

---

## Output

**Agenda Mode** produces a structured agenda with:
- Meeting purpose (one line)
- Two to three decisions to make, each with an owner
- Timed walk-through items
- What each attendee needs to bring

**Follow-Up Mode** produces:
- A recap email with a summary, decisions, action items (each with owner and
  due date), and open questions
- A clean action checklist to paste into your task tool
