# Due Diligence

Private equity due diligence setup and analysis planner. Sets up the complete workplan for any investment transaction, organizes available evidence across 26 diligence sections, identifies data gaps, and produces IC-oriented output.

Built by MBL Partners (https://mblpartners.com) -- part of the Thrive with AI skill library.

---

## What It Does

Takes any investment materials -- a CIM, teaser, financial model, or data room -- and establishes the full diligence process in one pass.

It does not summarize documents. It organizes evidence, classifies every data point by confidence level (verified fact, management claim, calculated value, or missing information), identifies what is missing and why it matters, builds a prioritized request tracker, flags red flags, and recommends the analyses the deal team should run next.

Covers all 26 core diligence sections (A through Z): transaction overview, investment thesis, financial performance, quality of earnings, revenue quality, customer diligence, product, market, competitive positioning, sales, operations, supplier, working capital, capex, forecast, management, human capital, technology, legal, tax, regulatory, ESG, insurance, debt, M&A strategy, and exit.

Adapts automatically to transaction type (platform, add-on, LBO, growth equity, carve-out, founder-owned) and industry (software, healthcare, consumer, industrial, business services, financial services, distribution).

## How to Run It

Once installed, trigger it by saying any of the following:
- "Start diligence on [company]"
- "Set up DD for [deal name]"
- "Run due diligence"
- "Build a diligence workplan"
- "Review this CIM"
- "Analyze this deal"
- "Prep for IC"

Drop in any available materials and the skill takes it from there.

---

## Output

A structured diligence package including:
- Executive diligence dashboard
- Preliminary investment thesis matrix
- Source inventory
- Extracted data by diligence section
- Data gap and request tracker (prioritized Critical through Post-close)
- Data reconciliation log
- Preliminary findings (positive, neutral, concerns, red flags, opportunities)
- Recommended analyses
- Management questions ordered by priority
- Specialist workstream recommendations
- Valuation and structure implications
- Next-step workplan (next 24 hours through post-close)
- Senior review checklist
